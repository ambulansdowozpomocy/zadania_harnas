Tagi blokowe czyli takie, które przenoszą treść do nowego wiersza:
  • <address></address> sekcja dla informacji kontaktowych
  • <article></article> sekcja dla artykułu
  • <aside></aside> sekcja dla treści pobocznej
  • <blockquote></blockquote> tag do cytatów
  • <canvas></canvas> element płótna
  • <dd></dd> opis elementu listy opisowej
  • <div></div> element podziałowy
  • <dl></dl> element listy opisowej
  • <dt></dt> nazwa elementu opisywanego w liście opisowej
  • <fieldset></fieldset> ugrupowanie elementów w formularzu
  • <figcaption></figcaption> definiuje podpis dla elementu figure
  • <figure></figure> dodaje opis dla obrazka na stronie
  • <footer></footer> najniższa sekcja strony
  • <form></form> element formularza
  • <h1></h1> największy nagłówek
  • <h2></h2> jeszcze więksiejejszy nagłówek
  • <h3></h3> więksiejszy nagłówek
  • <h4></h4> większy nagłówek
  • <h5></h5> nagłówek
  • <header></header> sekcja nagłówka strony
  • <hr></hr> linia horyzontalna podziałowa
  • <li></li> podpunkt z listy
  • <main></main> główna sekcja strony
  • <nav></nav> sekcja nawigacyjna strony
  • <noscript></noscript> wyświetla tekst w wypadku gdzie strona jest załadowana bez javascriptu
  • <ol></ol> znacznik listy numerycznej
  • <p></p> znacznik tekstu paragrafowanego
  • <pre></pre> znacznik wyświetlający zawarty tekst, bez zmiany formatowania
  • <section></section> sekcjowa sekcja strony
  • <table></table> znacznik tabeli
  • <tfoot></tfoot> znacznik podsumowującego rzędu tabelki
  • <ul></ul> znacznik listy kropkowej
  • <video></video> znacznik elementu wideo

  1. <a></a> znacznik linków

  3. <b></b> znacznik pogrubiający tekst
  4. <bdo></bdo> znacznik definiujący kierunek tekstu
  5. <button></button> guzik
  6. <cite></cite> stylizator tekstu na cytat
  7. <code></code> sekcja z kodem
  8. <dfn></dfn> pochyla tekst, i deklaruje że będzie definiowany
  9. <em></em> pochyla tekst, i zwiększa jego znaczenie
  10. <i></i> znacznik pochylający tekst
  11. <img></img> znacznik elementu obrazka
  12. <input></input> znacznik elementów wprowadzających dane
  13. <kbd></kbd> znacznik deklarujący klawisze
  14. <label></label> podpis dla elementów wprowadzających dane
  15. <link></link> znacznik definiujący zapożyczone zasoby
  16. <map></map> element mapujący strefy obrazka, do przekierowań
  17. <meta></meta> znacznik definiujący metadatę dokumentu
  18. <object></object> element zewnętrzny
  19. <output></output> element z rezultatem skryptu
  20. <q></q> znacznik krótkiego cytatu
  21. <samp></samp> definiuje przykładowy tekst
  22. <script></script> oznaczenie zawartości jako javascript
  23. <select></select> element listy wyboru
  24. <small></small> formatuje tekst na mniejszy
  25. <span></span> znacznik zmieniający kolor tekstu
  26. <strong></strong> znacznik pogrubiający tekst i nadający mu większe znaczenie
  27. <style></style> oznaczenie zawartości jako css
  28. <sub></sub> formatuje tekst jako subtekst
  29. <sup></sup> formatuje tekst jako supertekst
  30. <textarea></textarea> element wielowierszowego wprowadzania tekstu
  31. <time></time> znacznik definiujący czas i datę
  32. <var></var> definiuje zawartość jako zmienną
